Download it - Kodi video downloader

Download it is a powerful video downloader for Kodi. Features include:

  • Intuitive launch from the context menu
  • Select the streams you want, or use a sensible default selection
  • Modify the download URL, for example to download a higher resolution
  • Can run in background
  • See download progress
  • Powerful FFmpeg backend
  • Fallback to youtube-dl

The use of the add-on could be illegal depending of the stream and the country you live in. Please check before you use it.

Download it was originally developed by L0RE.

Download

The source code is available here.

In addition, installable .zip files and a repository are provided. The repository will automatically install new versions.

stable unstable
pre-Matrix: $STABLE_VERSION $UNSTABLE_VERSION nightly
Matrix: $STABLE_VERSION $UNSTABLE_VERSION nightly
Repository: $STABLE_REPO_VERSION $UNSTABLE_REPO_VERSION

You will also need to install FFmpeg. On Debian (incl. Raspbian) and Ubuntu you can typically just do:

sudo apt-get install ffmpeg

See also: FFmpeg download page.
